## 🚚 The Complete WooCommerce Delivery Drivers Solution

**Local Delivery Drivers for WooCommerce** helps WooCommerce stores manage their full local delivery workflow from one place.

Add delivery drivers, assign WooCommerce orders, update delivery statuses, let drivers manage orders from a mobile-friendly driver panel, track deliveries in real time, optimize routes, collect proof of delivery, and keep customers updated with email, SMS, and WhatsApp notifications.

Whether you run a restaurant, grocery store, pharmacy, florist, local courier service, furniture store, or any business that delivers locally, this plugin gives you the tools to manage WooCommerce delivery drivers more professionally.

Built to help WooCommerce stores scale their delivery operations, the plugin includes a modern admin and driver experience, advanced route management, delivery reports, driver applications, driver reviews and ratings, interactive maps, and automatic nearest-driver assignment.

## 🆓 Free Features

### Delivery Driver User Role

Add drivers to your WordPress site using the dedicated **Delivery Driver** role. Manage driver accounts, availability, contact details, and assigned orders.

### Delivery Order Statuses

The plugin adds delivery-specific WooCommerce order statuses:

* **Driver Assigned** — A driver has been assigned to the order.
* **Out for Delivery** — The order is on the way to the customer.
* **Failed Delivery Attempt** — The delivery could not be completed.

### Assign Drivers to WooCommerce Orders

Assign a delivery driver directly from the WooCommerce order page. Once assigned, the order status is automatically updated to **Driver Assigned**.

### Automatic Order Notes

Driver actions and delivery updates are automatically saved in WooCommerce order notes, giving store owners a clear delivery history for each order.

### Mobile-Friendly Driver Panel

Drivers get a dedicated mobile-friendly workspace designed to help them manage deliveries from any device.

The driver panel includes delivery dashboards, order management, customer and delivery information, delivery statuses, navigation tools, communication options, delivery history, driver settings, and everything drivers need to complete deliveries efficiently while on the road.

Drivers can view assigned deliveries, access detailed order information, contact customers, follow delivery routes, update delivery statuses, and manage their daily delivery workflow from a single interface.



### Supported Languages

English · Spanish · French · Hebrew · Portuguese · Russian · German · Italian · Arabic

RTL is supported.

== Installation ==

1. Back up your website and database before installation.
2. Download the plugin from the WordPress repository or from [powerfulwp.com](https://powerfulwp.com).
3. Upload the plugin to the `wp-content/plugins/` directory, or install it from **Plugins > Add New** in your WordPress dashboard.
4. Activate the plugin.
5. After activation, the **Delivery Driver** menu will appear in your WordPress admin.

### Important: Set Your WooCommerce Store Address

Before using the plugin, make sure your WooCommerce store address is configured correctly.

1. Go to **WooCommerce > Settings**.
2. Confirm that your store address fields are filled in correctly.
3. Save your changes.

This helps the plugin calculate routes, locations, and delivery information correctly.

== Frequently Asked Questions ==

= Is this plugin easy to use? =

Yes. The plugin is designed to be simple for store owners and drivers. Store owners manage drivers and orders from the WooCommerce admin, while drivers use a clean mobile-friendly panel.

= How do I add a new delivery driver? =

1. Go to **Delivery Drivers > Drivers**.
2. Click **Add Driver**.
3. Enter the driver's details.
4. Save the driver.
 
The driver can then log in to the driver panel and start receiving assigned deliveries.


= How do I assign a delivery driver to an order? =

Open the WooCommerce order page. In the **Delivery Driver** section, select the driver and update the order. The order status will change to **Driver Assigned**.

= Can I bulk assign orders to drivers? =

Yes. Bulk assignment is available in the premium version.

= Can drivers claim orders themselves? =

Yes. In the premium version, store owners can allow drivers to claim available orders directly from the driver panel.

= Does the plugin support route optimization? =

Yes. Route optimization is available in the premium version. Drivers can optimize routes by distance and navigate using Google Maps, Waze, or Apple Maps.

= Does the plugin support real-time delivery tracking? =

Yes. Real-time delivery tracking is available in the premium version. Customers can view the driver's live location, ETA, driver details, and delivery progress on a tracking page.

= Can customers see driver information? =

Yes. In the premium version, customers can see driver details such as driver name, photo, vehicle details, ETA, and call button.

= Can drivers upload proof of delivery? =

Yes. The premium version supports proof of delivery with signature capture and delivery photo uploads. Multiple delivery photos are supported.

= What notification channels are supported? =

The plugin supports email notifications. The premium version supports SMS and WhatsApp notifications using Twilio or the built-in PowerfulWP SMS provider.

= Does the plugin support multivendor marketplaces? =

Yes. The premium version supports Dokan, WCFM, and WCMP/MultiVendorX, including vendor pickup addresses, vendor phone numbers, and navigation to vendor locations.

= Does the plugin support driver commissions? =

Yes. Driver commissions are available in the premium version, including fixed, distance-based, time-based, distance + time, and fixed + distance commission options.

= Does the plugin include delivery reports? =

Yes. The premium version includes reports for driver commissions, sales by city, daily delivery trends, failed deliveries, and driver reviews.

= Does the plugin support driver applications? =

Yes. The premium version includes a driver applications system where drivers can apply from your website and admins can approve or reject applications.

= Does the plugin support driver reviews and ratings? =

Yes. The premium version includes customer reviews and ratings for drivers, including analytics and notifications.

= Does this plugin work on mobile? =

Yes. The driver panel is fully mobile-friendly and can be used from a smartphone browser. A native mobile app add-on is also available.

= Is there a native mobile app for drivers? =

Yes. The App for Delivery Drivers add-on provides a native Android and iOS app experience with push notifications, background tracking, and better mobile reliability.

= What languages does the plugin support? =

English, Spanish, French, Hebrew, Portuguese, Russian, German, Italian, and Arabic. RTL is supported.

= Is the plugin compatible with WooCommerce HPOS? =

Yes. The plugin supports WooCommerce High-Performance Order Storage, also known as HPOS.
